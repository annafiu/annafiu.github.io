# img folder

Taruh semua foto/gambar kamu di sini.

## File yang dibutuhkan website:

| Nama file          | Dipakai di         | Ukuran ideal     |
|--------------------|--------------------|------------------|
| `about-bg1.jpg`    | Home (foto profil) + About page  | min 800×800px    |
| `contact-photo.jpg`| Contact page       | min 900×600px    |

## Tips:
- Format: `.jpg` atau `.webp` (lebih ringan)
- Kompres dulu di: https://squoosh.app
- Nama file harus persis sama (huruf kecil, pakai dash)
- Untuk foto project/portfolio: bisa ditambah kapan saja nanti
